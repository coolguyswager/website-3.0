Hardware



In each computer there are sevaral pieces of hardware. The main ones are:

  • Motherboard
  • CPU
  • RAM
  • Hard drive
  • Power
  • Keyboard, Mouse, Monitor

The motherboard is like the postal service in which both of them take in information and send it out.

CPU stand for central processing unit. The CPU follows the instructions of programs. It process them and then carries it out.

RAM stands for random access memory. It can be somewhat described as multitasking memory. It allows you to do many many things all at once.

The hard drive stores all of your data. It is where all of your folders are kept. All of folders and files you make or download go there.
