HTML which stands for Hyper-text Mark-up Language is used to format webpages. It is important because without it we would not have a language to create the skeletons of a website .

CSS(Cascading Style Sheets) is the language that is paried with html and similar to HTML, CSS is used to format the text, layout, images, links, and everything else on websites.

    Tags are another tool used in html. They are used to help formate websites and they are defined with < >.



  • div (Allows you to edit everything inside of the div. If you only want to edit a single sentence or a paragraph you could put it into a div)
  • P (used to make a paragrpah)
  • h1, h2, h3 etc (diffrent headers for diffrent sizes)
  • Span (Similar to div but span is more commonly used to edit a word or phrase inside of a p tag)
    Attributes go along with tags to add on to them. For instance if you are using the p tag you could add the attribute title. Example: p title="title" There are a lot of other attributes like this and some are:
  • href
  • width
  • Height
